# Description
A Python script for export draw and hand write notes from images removing the background from the image.

Instructions
===
Run the follow command.
 ``` python draw_export.py filename.png ```
 A preview windows appears, use your mouse and select the area of interest.
 Press Space or Enter to export the store the selected area. You can choose multiple areas.
 After you complete the selections just press ESC two time.
 The script creates the folder output in your current directory, inside that folder you can find
 the selected areas in diferent png files with transparent background.
